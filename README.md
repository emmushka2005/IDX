# 🏡 IDX Exchange — MLS Analytics Project

An end-to-end real estate analytics pipeline built on live MLS transaction data. Covers data ingestion, aggregation, cleaning, mortgage rate enrichment, feature engineering, and outlier detection to produce reliable, analysis-ready housing market datasets.

---

## 🛠️ Tools & Technologies

| Category | Tools |
|---|---|
| **Languages** | Python |
| **Libraries** | Pandas, Matplotlib, Seaborn |
| **Visualization** | Tableau Desktop |
| **Data Sources** | CoreLogic Trestle API, FRED MORTGAGE30US API |
| **File Formats** | CSV, .twbx |

---

## ⚠️ Data Confidentiality

All MLS datasets are sourced from live transaction records provided by IDX Exchange and are **confidential**. Raw data files are excluded from this repository and must not be shared externally.
